# 📝 Task Tracker MVP

Консольное приложение для управления задачами. 

## 🎯 Возможности

- ➕ Добавление новых задач
- 📋 Просмотр списка всех задач
- ✅ Отметка задач как выполненных
- 💾 Автоматическое сохранение в JSON файл

## 🚀 Как запустить

```bash
git clone https://github.com/SkyMage0/task-tracker-mvp.git
cd task-tracker-mvp
python tasks.py
