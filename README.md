# 📝 Task Tracker

Веб-приложение для управления задачами с приоритетами, дедлайнами и статистикой.

## ✨ Возможности

- ➕ Добавление задач с приоритетом (🔴 высокий / 🟡 средний / 🟢 низкий)
- 📅 Установка дедлайнов
- 📋 Просмотр задач (сортировка по статусу и приоритету)
- ✅ Отметка выполнения
- 🗑️ Удаление задач
- 📊 Статистика и графики
- 🎨 Современный веб-интерфейс
- 🔄 JSON API для интеграций

## 🚀 Как запустить

### Консольная версия
```bash
python tasks.py
Веб-версия
bash
python web_app.py
Затем открой браузер и перейди на http://localhost:5000



🛠 Технологии
Компонент	Технология
Бэкенд	Python 3.x
Веб-фреймворк	Flask
Хранилище	JSON
Фронтенд	HTML5, CSS3
Шаблонизация	Jinja2
Тестирование	Pytest
📊 API Endpoints
Метод	URL	Описание
GET	/	Главная страница
GET	/statistics	Страница статистики
GET	/api/tasks	JSON список задач
POST	/add	Добавить задачу
GET	/complete/<id>	Выполнить задачу
GET	/delete/<id>	Удалить задачу
