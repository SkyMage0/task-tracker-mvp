# 📝 Task Tracker

Веб-приложение для управления задачами с приоритетами, дедлайнами и статистикой.

## ✨ Возможности

- ➕ Добавление задач с приоритетом (🔴 высокий / 🟡 средний / 🟢 низкий)
- 📅 Установка дедлайнов
- 📋 Просмотр задач (сортировка по статусу и приоритету)
- ✅ Отметка выполнения
- 🗑️ Удаление задач
- 📊 Статистика и графики
- 🎨 Современный веб-интерфейс
- 🔄 JSON API для интеграций

## 🚀 Как запустить

### Консольная версия
```bash
python tasks.py
Веб-версия
bash
python web_app.py
Затем открой браузер и перейди на http://localhost:5000
